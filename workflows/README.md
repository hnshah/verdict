## What You'll Learn

### Quick Start
- Install and configure Verdict
- Run your first eval
- Interpret results
- Common troubleshooting

### Model Comparison
- Compare 5+ models at once
- Analyze cost-quality tradeoffs
- Make data-driven routing decisions
- Set up baselines

### Regression Testing
- Detect quality drops before deploy
- Set up CI/CD integration
- Automate daily checks
- Define acceptance criteria

### Production Monitoring
- Log production requests
- Sample and evaluate continuously
- Set up dashboards and alerts
- A/B test new models safely

## Prerequisites

- Node.js 18+ installed
- Ollama running (for local models)
- API keys for cloud models (optional)
