## Contribution Workflow

**For each run:**
1. Create config in `configs/`
2. Run: `verdict run --config configs/runX.yaml`
3. Wait for completion (checkpoint saves progress)
4. Find result: `results/YYYY-MM-DD-HH-MM-SS.json`
5. Contribute: `verdict contribute --result results/xxx.json --author "Rak" --token $GH_TOKEN`
6. Verify on dashboard
