# Examples

These eval packs are for testing, demos, and scorer feature exploration — not production benchmarking.

| File | Purpose |
|------|---------|
| `quick-test.yaml` | Fast sanity check (2-3 cases) |
| `python-quick.yaml` | Quick Python smoke test |
| `tool-test.yaml` | Basic tool-calling smoke test |
| `tool-calling-advanced.yaml` | Advanced tool-calling patterns |
| `tool-edge-cases.yaml` | Tool-calling edge cases |
| `edge-case-marathon.yaml` | Broad edge case sweep |
| `extreme-edge-cases.yaml` | Adversarial / stress inputs |
| `stress-test.yaml` | Performance under load |
| `fuzzy-match-examples.yaml` | Fuzzy scorer demos |
| `javascript-scorer-examples.yaml` | JavaScript scorer demos |
| `multi-assertion-examples.yaml` | Multi-assertion scorer demos |
| `system-prompt-examples.yaml` | System prompt override demos |
| `elite-quick-2cases.yaml` | 2-case elite test artifact |
| `elite-test-single.yaml` | Single-case elite test artifact |

For canonical benchmarking packs, see the parent `eval-packs/` directory.
