## Description

From issue:
> PEP 810 "Explicit lazy imports" has been accepted and is coming to Python 3.15. This adds a new `lazy` keyword for imports and `__lazy_modules__` for backwards compatibility.

**New syntax:**
```python
lazy import json
lazy from json import dumps
```

**Backwards compatibility:**
```python
__lazy_modules__ = ["json"]
import json
```

## Technical Approach

1. **Parse new syntax:**
