Here is a SQL query that finds the top 3 customers by total order value from tables orders and customers: ``` SELECT c.name, SUM(o.amount) AS total FROM orders o JOIN customers c ON o.customer_id = c.id GROUP BY c.id ORDER BY total DESC LIMIT 3; ``` This query joins the orders and customers tables on the customer_id column, and then groups the results by customer ID. The SUM function is used to calculate the total amount for each customer. The results are then ordered by the total amount in descending order and limited to the top 3 customers. Note that this query assumes that the orders table has a customer_id column that references the id column in the customers table. If your tables have different column names, you will need to adjust the query accordingly.
