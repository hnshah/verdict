To implement a basic LRU (Least Recently Used) cache in Python with `get(key)` and `put(key, value)` methods, we can use a combination of a dictionary and an ordered list. Here's a step-by-step implementation: 1. **Dictionary**: This will store the key-value pairs. 2. **Ordered List**: This will store the keys in the order they were accessed. The most recently used key will be at the end of the list, and the least recently used key will be at the beginning. We will use Python's `collections.OrderedDict` to simplify the implementation of the ordered list. Here's the complete implementation: ```python from collections import OrderedDict class LRUCache: def __init__(self, capacity: int): self.cache = OrderedDict() self.capacity = capacity def get(self, key: int) -> int: if key not in self.cache: return -1 else: # Move the key to the end to show that it was recently used self.cache.move_to_end(key) return self.cache[key] def put(self, key: int, value: int) -> None: if key in self.cache: # Move the key to the end to show that it was recently used self.cache.move_to_end(key) self.cache[key] = value if len(self.cache) > self.capacity: # Remove the first key (least recently used) self.cache.popitem(last=False) # Example usage: cache = LRUCache(2) cache.put(1, 1) cache.put(2, 2) print(cache.get(1)) # returns 1 cache.put(3, 3) # evicts key 2 print(cache.get(2)) # returns -1 (not found) cache.put(4, 4) # evicts key 1 print(cache.get(1)) # returns -1 (not found) print(cache.get(3)) # returns 3 print(cache.get(4)) # returns 4 ``` ### Explanation: 1. **Initialization (`__init__` method)**: - `self.cache`: An `OrderedDict` to store the key-value pairs. - `self.capacity`: The maximum number of items the cache can hold. 2. **Get Operation (`get` method)**: - If the key is not in the cache, return `-1`. - If the key is in the cache, move the key to the end of the `OrderedDict` to mark it as recently used and return its value. 3. **Put Operation (`put` method)**: - If the key is already in the cache, move it to the end of the `OrderedDict` to mark it as recently used. - Update the value of the key in the cache. - If the cache exceeds its capacity, remove the first key (the least recently used) from the `OrderedDict`. This implementation ensures that both `get` and `put` operations are O(1) due to the properties of the `OrderedDict`.
